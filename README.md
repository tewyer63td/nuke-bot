 ## Usage
 
 1. Download the code.
 2. Go to index.js
 3. Replace `your-token-goes-here` by your Discord APP <b>Token</b> (at the end).
 4. Replace `nuke-message` by your Nuke message.
 5. Open a CMD.
 6. CD into the project code.
 7. Run `node index.js`
 8. Run !nuke in a <b>Test Server</b>!
